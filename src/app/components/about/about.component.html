Unsplash Dev. API

A random image is fetched from the Unsplash API using the Javascript Reactive Extensions (RxJS) library to sbuscribe to an API JSON data Observable

The data on the Contact page is fetched from the Github API. No API key is required to search for a user. For more detailed info. an API key is required. 6

The Stack: Angular 11, Tailwindcss 2, RxJS
Testing: Jasmine, Karma
