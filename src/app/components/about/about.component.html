Unsplash API

A random image is fetched from the Unsplash API using the Javascript Reactive Extensions (RxJS) library to subscribe to an API JSON data Observable

The data on the Contact page is fetched from the Github API. No API key is required to search for a user. For more detailed info. an API key is required.

The Stack: Angular 17, TailwindCSS 3, RxJS 7
Testing: Jasmine, Karma
